# GrowthApp
This is an Android app inspired by "Atomic Habits" by James Clear and "Tuesdays with Morrie" by Mitch Abom. It is designed to promote personal growth through small, consistent changes, helping users build habits, set meaningful goals and reflect on their progress.

## Features
- Set 2 core goals along with 2 smaller habit goals that you wish to incorporate into your lifestyle
- Reflect on achievements and growth through the 'End of Day Reflection' feature
- Set a goal duration for each habit, and the app will progressively increase the amount of time you spend on it each day until the target is reached
- Track daily progress through your Dashboard
- Participate in 'Heart Walks' where you attempt to find heart-shaped patterns in your surroundings, such as leaves, clouds or water stains. This practice encourages mindfulness, intentional focus, and awareness of how our attention shapes what we notice in everyday life.


## Technologies Used
- Java
- XML
- Android Studio
